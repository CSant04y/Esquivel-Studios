# Esquivel Studios

A website for Esquivel Studios, a portrait and candid photography business based in Tulsa, Oklahoma.

## Overview

This is a single-page website featuring:

- **Hero Section** - Full-screen introduction with call-to-action
- **Portfolio Gallery** - Responsive grid showcasing photography work
- **About Section** - Background information about the photographer
- **Booking Form** - Session request form with availability information
- **Contact Section** - Contact details and location

## Files

- `EsquivelStudios.html` - Main HTML file
- `HomepageStyles.css` - Stylesheet with responsive design

## Features

- Smooth scroll navigation
- Responsive design for mobile and desktop
- Interactive booking form
- Hover animations on gallery items and buttons

## Usage

Open `EsquivelStudios.html` in a web browser to view the site.

- Both Portrait & Candid
